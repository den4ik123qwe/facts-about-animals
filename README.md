После запуска программы появляется GUI-интерфейс в котором вы можете выбрать город интересующий вас, слева есть области из базы данных на выбор, после нажатия на интересующую вас область сверху рисунка животного у вас появляется список животных из этого региона. После выбора вами интересующего вас животного вы можете увидеть его рисунок и почитать интересный факты про него
